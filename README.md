## ğŸš€ Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn
- Expo Go app on your device (optional)
- Supabase account (for backend)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/mohitbansal25082006/StudyBuddy.git
   cd StudyBuddy
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Set up environment variables**
   ```bash
   cp .env.example .env
   ```
   Then fill in your Supabase and OpenAI API keys in the `.env` file.

4. **Set up Supabase**
   - Create a new project in Supabase
   - Run the SQL migrations in the `migrations` folder
   - Set up storage buckets for avatars and post images
   - Configure authentication providers

5. **Run the app**
   ```bash
   npm start
   ```

6. **Run on device**
   - Install Expo Go on your device
   - Scan the QR code shown in the terminal
   - Or run on a simulator/emulator
